# 🌟 BackEnd Developer - 황성현

안녕하세요! Java와 Spring Boot를 주력으로 사용하여, **안정적인 시스템 구축과 레거시 코드의 성능 최적화**에 기여하는 개발자입니다. 특히 **리팩토링을 통한 데이터 처리 효율 개선 및 유지보수성 향상**을 엔드 투 엔드(End-to-End)로 주도한 경험이 있습니다.

---

## 💡 핵심 역량 (Core Competencies)

* **Java, Spring, SQL, ERD 설계, 도메인 모델링, JPA**
* **레거시 시스템 성능 개선, 코드 리팩토링, 아키텍처 설계 이해**
* 사용자 중심 개발: 현직 간호사와 소통하며, 품질 관리·일정 조율·요구사항 반영

---

## 🚀 주요 프로젝트 및 경험 (Projects & Experience)

### 1. 청구 시스템 리팩토링 (동은정보기술, 2025)

* 10,000줄 이상 복수 쿼리를 비즈니스 로직과 분리·최적화하여 **데이터 처리 속도 53% 향상**
* 전체 구조 재설계 및 핵심 로직 구현 주도 (기존 100명분 공단 청구 파일 생성 시간 5분 소요를 **1~2분대**로 단축)
* 책임과 역할을 명확하게 구별하여 **유지보수성**을 높임 (기존 로직 디버깅의 어려움 개선, 수정해야 할 부분이 한 눈에 파악 가능)

### 2. "Insty" 백엔드 개발 (진행 중: 2025.08.30~)

* **Redis TTL**을 이용한 토큰 기반 사용자 검증 및 인증 API 구현
* **멀티모듈 아키텍처** 기반, **TDD**와 **DDD**를 적용하여 프로젝트 구성
* 강좌와 관련된 API 구현

### 3. "학사모" 앱 개발 (대학교 팀 프로젝트, 백엔드 리더)

* **Redis**를 이용한 **Pub/Sub 구조**로 채팅 서비스 구현
* 도메인 모델·**ERD 설계**, CRUD API 구축
* 팀원 간 역할 분배 및 일정 관리, 품질 검증 프로세스 운영 주도
